概要：
	本フォルダに含まれているファイルは, 
	EyeBotを8の字走行させるプログラム"figure-of-eight.hex"のソースコードです。
	動作確認に用いたEyeBotは, 早稲田大学理工学部 電気工学実験室が保有するEyeBotです。

各ファイルの意味：
	main.c = EyeBot起動時に使用する関数と、画像解析関数を含んだソースファイル
	mode.c = 各モードで使用する関数のソースファイル
	sys.c  = LCDに変数表示する関数や、サーボのoffsetを調整する関数のソースファイル
	main.h = 上記の3つのファイルに対するヘッダーファイル
	wabotcam.c = (実験テキストと同一のもの)
	wabotcam.h = (実験テキストと同一のもの)

コンパイル方法：
	gcc68が実行できるコンピュータにおいて, cmd.batを実行しコンソールを立ち上げます。
	その後, コンソール画面上で"make"と打ち込んで, make.batを実行します。
	コンパイルに成功した場合, カレントディレクトリに"figure-of-eight.hex"が生成されます。
	
	※なお, コンパイラは下記サイトからダウンロードできます。
	http://robotics.ee.uwa.edu.au/eyebot/ftp/
